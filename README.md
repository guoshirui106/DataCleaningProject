# Getting and Cleaning Data Project
This is the course project for Getting and Cleaning Data.

## Files
- run_analysis.R: R script for data cleaning
- tidy_data.txt: Final tidy dataset
- CodeBook.md: Description of variables

## Steps
1. Merge training and test data
2. Extract mean and std columns
3. Name activities descriptively
4. Rename variables
5. Create average data by subject and activity
